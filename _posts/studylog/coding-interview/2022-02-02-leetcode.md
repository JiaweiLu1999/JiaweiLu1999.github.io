    - Binary Search
    - Find left most: `left = mid + 1, right = mid`
        ```python = 
        while left < right:
            mid = (left + right) // 2
            if nums[mid] < target:
                left = mid + 1
            else:
                right = mid
        ```
    - Find right most: `left = mid, right = mid - 1`
        ```python = 
        while left < right:
            mid = (left + right + 1) // 2
            if nums[mid] > target:
                right = mid - 1
            else:
                left = mid
        ```

39. Combination Sum
    - Backtracking/DFS
    ```python
    def dfs(nums, target, path, res):
            if target < 0:
                return
            elif target == 0:
                res.append(path)
            else:
                for i in range(len(nums)):
                    dfs(nums[i:], target - nums[i], path + [nums[i]], res)
     ```
